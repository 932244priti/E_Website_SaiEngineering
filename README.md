# 🛒 E-Commerce Website (Sai Engineering)

## 📌 Project Description
This project is a dynamic **E-Commerce Website for Carbide Tools** developed using web technologies.  
It allows users to browse products, add items to cart, and complete purchases with a secure login system.

The system is designed to provide a **smooth and user-friendly shopping experience**.

---

## 🎯 Objectives
- To develop a functional e-commerce platform  
- To implement user authentication (Login/Register)  
- To manage products and cart system  
- To understand frontend-backend integration  

---

## 🛠️ Technologies Used

### 💻 Frontend
- HTML  
- CSS  
- JavaScript  

### ⚙️ Backend
- PHP  

### 🗄️ Database
- MySQL  

---

## ✨ Features
- 👤 User Registration & Login  
- 🛍️ Product Catalog  
- ➕ Add to Cart  
- 🧾 Checkout System  
- 💳 Payment Options  
- 📞 Contact Page  
- 🔐 Secure Authentication
